# lamp_portfolio
私のポートフォリオです。
<p class="mb-5 text-left">エンジニア向けにオススメの参考書やPCを販売するデモECサイトを作成しました。<br>以下の機能を備えています。</p>
<ul class="text-left">
    <li>ログイン機能</li>
    <li>ユーザー登録機能</li>
    <li>商品管理機能</li>
    <li>ショッピングカート機能</li>
    <li>購入機能</li>
    <li>注文履歴/購入明細書機能(テーブル設計/正規化)</li>
    <li>セキュリティ対応(SQLインジェクション、XSS、CSRF)</li>
    <li>ページング機能、並び替え機能</li>
</ul>
